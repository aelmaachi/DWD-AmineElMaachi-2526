# Agent Instructions - DWD Button Panel

## Project context
This is a Dynamic Web Development school project for an IT course at Odisee.
The project is a single page application made with vanilla HTML, CSS and JavaScript.

## General rules
- Use only vanilla HTML, CSS and JavaScript.
- Do not use frameworks or libraries such as React, Vue, Bootstrap or jQuery.
- Use clear and simple code that a beginner/intermediate IT student can explain.
- Use querySelector() and querySelectorAll() for DOM selection.
- Keep JavaScript organized in this order:
  1. constants and variables
  2. function declarations
  3. event handlers
  4. event listeners / initialization
- Use fetch with async/await for API requests.
- Add useful comments in the code.
- Do not make huge changes at once. Work feature by feature.
- Keep the design responsive.
- Do not add code that is impossible to explain during a project defense.

## Project requirements
The application must:
- Search sounds using the Freesound API.
- Show search results with sound information.
- Play a sound when clicking a sound button.
- Stop the sound when clicking the same button again.
- Stop the current sound when another sound starts.
- Visually show which sound is currently playing.
- Add sounds from search results to a dashboard.
- Remove sounds from the dashboard.
- Save dashboard sounds in localStorage.

## Code style
- Use meaningful variable and function names.
- Avoid duplicated code where possible.
- Keep functions small and focused.
- Comment important logic.
- Make sure the app does not crash during normal use.
